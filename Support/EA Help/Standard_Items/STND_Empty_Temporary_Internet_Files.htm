Empty Temporary Internet Files

Whenever you browse the Internet and visit a web site Windows stores the files from that site in a temporary location so that the next time you visit that site it can load quicker.  However, over time the amount and size of these files can become very large.  If a game needs to make use of this temporary location it may not be able to.  Emptying this temporary folder will free up this resource for any online applications that might need it.  To do so:

 

1.  Open your Windows Control Panel and then open Internet Options.

2.  Click the Delete Files button, check the box labeled 'Delete all offline content' and then click OK.
