Check Minimum Requirements

PC games often require the latest in hardware technology to run.  The game box should list what these requirements are and you can find out what your PC has and if it meets these requirements through a number of methods.

  • Check System Requirements Yourself.

    To check the System Requirements you can do the following:
    1. Left click on the START button and choose RUN
    2. In the dialog box that appears type the word DXDIAG then click OK
    From this application you can see all of the components that are installed on your machine. The ones you are looking for are:
    • Operating System
    • Video and Sound Devices
    • Processor
    • Memory
    • Direct X version
