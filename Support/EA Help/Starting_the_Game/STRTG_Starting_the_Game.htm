Issues Starting the Game

If you are having issues when starting the game use the link below that best describes the issue you are having.  These are the troubleshooting tasks that EA Technical Support Reps use to solve the majority of these types of issues.
Errors Related to DirectX: DirectSound, Direct3D, DirectDraw, DirectInput, etc
CD Errors: Insert Correct CD, Can't Find CD, etc
All Other Errors
Crashing Back To Desktop
Locked Up: Game locks up after it is launched

 

 

EA Tech Tip:

To prevent possible resource conflicts, always close background tasks before playing the game.
