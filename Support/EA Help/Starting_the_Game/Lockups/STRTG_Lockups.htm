Lockups Starting the Game When Starting The Game

If your game is locking up or fails to respond after it is launched follow the guide below that best describes the situation.

Copy Protection Lockup:  The game locks up before even entering a full screen mode.  A title screen is shown for a moment and then the game disappears.
Locking Up At A Blank Screen:  The game goes to a full blank screen and stops responding.
Locking Up With Repeating Sound:  The game is locked up but the sound continues to repeat

For all other types of lockups check our Tech Support web site at http://techsupport.ea.com

 

 

 

EA Tech Tip:

To prevent possible resource conflicts, always close background tasks before playing the game.
