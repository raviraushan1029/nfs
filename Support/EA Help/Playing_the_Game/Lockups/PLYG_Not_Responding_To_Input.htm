Not Responding To Input

If the game appears to still be running (e.g. the background continues to move, music is still playing etc.) but you cannot interact with it any longer follow the steps below.  

 

Close Background Tasks

 

Try reinstalling the game.  Uninstall the game and reinstall it.  It may be some trouble with the game files.

 

Check our Tech Support web site for any known issues.

 

 

EA Tech Tip:

If you are using wireless devices make sure that the receiver is not too close to other computer equipment and other wireless devices.  Also make sure that the batteries are charged.
