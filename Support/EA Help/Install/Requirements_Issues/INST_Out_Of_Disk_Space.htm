Out Of Disk Space

It is important to make sure that you have enough hard disk space available on the hard disk that you want to install the game to.   You will need to make sure that not only do you have enough hard disk space to install the game, but also enough space for DirectX DirectX is an application programming interface created by Microsoft. DirectX allows programmers to make a game that will work on any windows computer without having to know about the specific hardware that is in the computer. EA Games require that the correct version of DirectX be installed on the computer, usually the latest version available. DirectX requires that video and sound drivers that match the version being used are also installed on the computer. and Windows Virtual Memory Virtual memory is hard drive space that Windows sets aside as cache.  Windows will be able to offload items from main RAM so that it can still be accessed by the program again..  Use the formula below to make sure you have enough free hard disk space.

 

Total Hard Disk Space needed = Game Install + DirectX + Windows Virtual Memory (Windows 98/ME only)

Check Virtual Memory settings.
Other things to try:
If you are low on Hard disk space you may want to try one of the following:
> Remove unused software from your machine.
> Use the Disk Cleanup Wizard to clear temp files.
> If the game offers it, try a different install size.  Check the game manual for availability of these.

 

 

 

EA Tech Tip:

Games will not reliably run on removable drives.  Also, you will want to make sure you are not running any CDROM enhancing utilities as these often conflict with the game and installations.
