  • Check System Requirements Yourself.

    To check the System Requirements you can do the following:
    1. Left click on the START button and choose RUN
    2. In the dialog box that appears type the word DXDIAG
    From this application you can see all of the components that are installed on your machine. The ones you are looking for are:
    • Operating System
    • Video and Sound Devices
    • Processor
    • Memory
    • Direct X version
