Installation Troubleshooting

The following section of this help troubleshooting manual will guide you through common problems that occur when installing our games.  Issues have been arranged into the following categories:

 

Pre-Install Preparation: Follow these steps before starting your install.

Requirements Issues:  Are you receiving prompts that your system does not have the hardware to run the game?  Are you receiving messages that you are missing something or are having a software conflict.
Media Issues: Is the CDROM not being read properly?  Are you being prompted to insert a disk?
Issues After Install: Windows looks different?  Game won't start?

 

 

EA Tech Tip:

The first step of any install problem is to make sure that there are no scratches or smudges on the disc.
